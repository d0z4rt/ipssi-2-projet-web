# Projet web 2026

## Folders

- `.husky` - git hooks
- `.vscode` - editor configs
- `apps` - application code
  - `client` - react frontend
  - `api` - express backend

## Installation

Install [pnpm](https://pnpm.io/)

### Dependencies

```sh
pnpm i
```

## Development

```sh
pnpm dev
```

## Build

```sh
pnpm build
```
